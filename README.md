## Project Features

- ### Responsive corporate website optimized for desktop, tablet, and mobile devices

- ### Modern UI/UX with smooth animations and interactive layouts

- ### Public-facing pages for Home, About, Services, Projects, Products, Clients, Team, and Contact

- ### Role-based authentication with login, registration, logout, forgot password, and reset password

- ### Protected admin dashboard for managing users, employees, clients, services, projects, products, and news

- ### CRUD functionality for business content with organized management panels

- ### Project and product listings with filtering, pagination, and detailed single-view pages

- ### Image upload system with preview, multiple image support, and client-side compression

- ### Contact form with validation and backend API integration

- ### SEO optimization with dynamic meta tags, canonical URLs, structured data, sitemap, and robots.txt

- ### API-driven architecture using Redux Toolkit and Axios for efficient data handling

- ### Light/dark mode support for improved user experience
